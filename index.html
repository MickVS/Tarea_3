Ejemplos del sistema Grid de Bootstrap

Estos son algunos arreglos básicos proporcionados por Bootstrap 5.

Tres columnas del mismo ancho

Consiga tres columnas de igual ancho con escala de desktops hasta large desktops. En dispositivos móviles y tabletas, las columnas se volverán una pila de forma automática.

Alternativa para tres columnas del mismo ancho

Usando las clases .row-cols-*, es posible crear un grid de tres columnas del mismo ancho de una manera más sencilla.

Tres columnas con anchos variables

Consiga tres columnas de ancho variable con escala de desktops hasta large desktops. Recuerde, las columnas de un grid siempre deben ser como máximo 12. Más que esto, y las columnas empezarán a apilarse sin importar el viewport.

Dos columnas con indentación

Indentar es sencillo, solo debe colocar una fila de columnas dentro de otra columna. Esto le da dos columnas con escala de desktops hasta large desktops, con otras dos columnas (de anchos iguales) dentro de la primera.

En dispositivos móviles y tabletas, las columnas principales y las internas se apilarán.
